# Modern Shopping Website (React + Firebase + FakeStore API)

## Tech Stack

* React.js
* React Router
* Firebase Authentication
* FakeStore API
* Context API
* CSS

---

# Overall Website Structure

```
Navbar
│
├── Logo
├── Search Bar
├── Categories Dropdown
├── Wishlist (Badge)
├── Cart (Badge)
└── Login / Profile

--------------------------------

Pages
│
├── Home
├── Shop
├── Product Details

├── Wishlist

├── Register

--------------------------------

Footer
```

---

# Navbar

### Purpose

Provide navigation and quick access to important features.

### Should Include

* Brand Logo
* Home
* Search Bar
* Categories Dropdown
* Wishlist Icon + Badge
* Cart Icon + Badge
* Login / Profile
* Logout (after login)
* Sticky on Scroll
* Responsive Mobile Menu

---

# Footer

### Purpose

Website information and useful links.

### Should Include

* Logo
* About
* Contact
* Privacy Policy
* Terms & Conditions
* Social Media Links
* Newsletter Subscription
* Copyright

---

# Home Page

### Purpose

Welcome users and highlight products.

### Sections

* Hero Banner
* Categories
* Featured Products
* Promotional Banner
* Why Choose Us
* Newsletter
* Footer

---

# Shop Page

### Purpose

Browse all products.

### Features

* Search
* Category Filter
* Sort
* Product Grid
* Load More Button
* Loading State
* Error State
* No Products Found State

---

# Product Card

Each product card should contain

* Product Image
* Product Title
* Category
* Rating
* Price
* Wishlist Button
* Add to Cart Button
* View Details Button

---

# Product Details

### Purpose

Display complete product information.

### Should Include

* Large Product Image
* Title
* Description
* Category
* Rating
* Price
* Quantity Selector
* Add to Cart
* Wishlist
* Related Products

> **Do not add Color or Size selectors.** FakeStore API does not provide this data, so adding fake options adds complexity without improving the project.

---

# Cart

### Purpose

Review selected products before checkout.

### Should Include

* Product List
* Quantity Increase
* Quantity Decrease
* Remove Item
* Clear Cart
* Coupon Input (UI only)
* Price Summary

  * Subtotal
  * Discount
  * Total
* Checkout Button
* Empty Cart State

---

# Wishlist

### Purpose

Save products for later.

### Should Include

* Saved Products
* Remove from Wishlist
* Move to Cart
* Empty Wishlist State

---

# Checkout

### Purpose

Complete the order.

### Should Include

* Shipping Information
* Delivery Address
* Payment Method (Demo)
* Order Summary
* Place Order Button

> No real payment gateway is required.

---

# Order Success

### Purpose

Confirm successful purchase.

### Should Include

* Success Illustration
* Success Message
* Order ID
* Delivery Estimate
* Continue Shopping Button

---
